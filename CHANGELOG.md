### Added

- **Yandex 360 Disk Node**: New node for interacting with Yandex 360 Disk storage
  - File operations: Upload, Download, Delete, Copy, Move, Get Info, Publish, Unpublish
  - Folder operations: Create, List, Delete, Get Info, Publish, Unpublish
  - Binary data support for file upload and download
  - Async operation handling with configurable wait-for-completion
  - Full TypeScript support with type-safe constants
  - Comprehensive error handling with actionable error messages
